### Инструкция по запуску приложения из исходного кода

1. Убедиться что установлен Python версии 3.6.8
2. Клонировать исходный код через

   ```Shell
   git clone https://github.com/username/image-processing-app.git
   ```
3. Установить зависимости

   ```Shell
   pip install -r requirements.txt
   ```
4. Запустить приложение

   ```Shell
   python main.py
   ```
